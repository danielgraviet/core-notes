# Tasks

Drop task files here to give agents focused instructions for a specific piece of work.

## Suggested format for a task file

**Filename**: `descriptive-task-name.md`

```markdown
# Task: <short title>

## Goal
What should be true when this task is done?

## Context
Any background the agent needs — why this matters, constraints, related files.

## Acceptance Criteria
- [ ] Specific, checkable outcomes
- [ ] ...

## Files Likely Involved
- `core-notes/SomeFile.swift`
```
